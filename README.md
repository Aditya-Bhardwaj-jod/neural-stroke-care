# 🚀 Aditya Bhardwaj — Portfolio

> Personal portfolio website built with React, Three.js, and GSAP.

🌐 **Live:** [aditya-bhardwaj.vercel.app](https://aditya-bhardwaj.vercel.app)

---

## ✨ Features

- 🎬 **Loading Screen** — GSAP letter-by-letter reveal animation
- 🌌 **Three.js Particle Field** — Interactive particles that react to mouse movement
- ⌨️ **Typing Animation** — Dynamic role switcher in Hero section
- 📜 **Scroll Animations** — GSAP ScrollTrigger on all sections
- 🖱️ **Custom Cursor** — Emerald glowing dot with ring follower
- 📊 **Stats Counter** — Animated number counter on scroll
- 💻 **Terminal Contact Form** — EmailJS powered terminal-style form
- 🎮 **Konami Code Easter Egg** — ↑ ↑ ↓ ↓ ← → ← → B A
- 🌙 **Dark / Light Mode** — Theme toggle
- 📱 **Responsive** — Works on all devices
- ⚡ **Smooth Scroll** — Lenis smooth scrolling
- 📈 **Scroll Progress Bar** — Top green progress indicator

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| React + TypeScript | Frontend framework |
| Three.js | 3D particle field |
| GSAP + ScrollTrigger | Animations |
| Lenis | Smooth scrolling |
| EmailJS | Contact form |
| Vite | Build tool |
| Vercel | Deployment |

---

## 📁 Sections

- **Hero** — Particle field + typing animation
- **About** — Code block + stats counter
- **Skills** — Tech categories with stagger animation
- **Experience** — Timeline with GSAP animations
- **Projects** — 3D tilt cards with live links
- **Certifications** — Google Coursera certificates
- **Contact** — Terminal style form

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Aditya-Bhardwaj-jod/Portfolio

# Install dependencies
cd Portfolio
npm install

# Run locally
npm run dev

# Build for production
npm run build
```
